Switch

 ·  Meerdere keuzes overzichtelijk

Wanneer gebruiken?

Een switch is handig als je één variabele vergelijkt met meerdere vaste waarden. Overzichtelijker dan een lange if/else-if keten.

Klassieke switch

int dag = 3;

switch (dag) {
    case 1:
        System.out.println("Maandag");
        break;
    case 2:
        System.out.println("Dinsdag");
        break;
    case 3:
        System.out.println("Woensdag");
        break;
    default:
        System.out.println("Onbekende dag");
        break;
}

Switch met String

String kleur = "rood";

switch (kleur) {
    case "rood":
        System.out.println("Stop!");
        break;
    case "oranje":
        System.out.println("Opgelet!");
        break;
    case "groen":
        System.out.println("Rijd!");
        break;
    default:
        System.out.println("Onbekende kleur");
}

Meerdere cases tegelijk

int maand = 4;

switch (maand) {
    case 12:
    case 1:
    case 2:
        System.out.println("Winter");
        break;
    case 3:
    case 4:
    case 5:
        System.out.println("Lente");
        break;
    default:
        System.out.println("Zomer of herfst");
}
⚠️Vergeet break; niet! Zonder break valt Java door naar de volgende case (fall-through).

While-lus

 ·  Herhalen zolang een voorwaarde geldt

While

De lus wordt uitgevoerd zolang de voorwaarde true is. Als de voorwaarde al meteen false is, wordt de lus nooit uitgevoerd.

int teller = 0;

while (teller < 5) {
    System.out.println("teller = " + teller);
    teller++;   // NIET vergeten, anders oneindige lus!
}
// Uitvoer: teller = 0, 1, 2, 3, 4

Do-while

De lus wordt minstens één keer uitgevoerd, daarna pas gecontroleerd.

int getal = 10;

do {
    System.out.println("getal = " + getal);
    getal--;
} while (getal > 0);

Break en continue

int i = 0;
while (i < 10) {
    i++;
    if (i == 3) continue; // sla 3 over, ga naar volgende iteratie
    if (i == 7) break;    // stop de lus volledig bij 7
    System.out.println(i);  // print 1, 2, 4, 5, 6
}
⚠️Vergeet de teller nooit te verhogen in een while-lus, anders krijg je een oneindige lus en bevriest je programma!

For-lus

 ·  Herhalen met een teller

Standaard for-lus

Bestaat uit drie delen: start, voorwaarde en stap.

// for (start; voorwaarde; stap)
for (int i = 0; i < 5; i++) {
    System.out.println("i = " + i);
}
// Uitvoer: i = 0, 1, 2, 3, 4

Achterwaarts tellen

for (int i = 10; i >= 0; i--) {
    System.out.println(i);
}
// 10, 9, 8 ... 1, 0

For-each lus (enhanced for)

Voor het doorlopen van arrays of lijsten, zonder index nodig:

String[] namen = {"Jan", "Piet", "An"};

for (String naam : namen) {
    System.out.println(naam);
}

Geneste for-lussen

// Tafel van vermenigvuldiging
for (int i = 1; i <= 3; i++) {
    for (int j = 1; j <= 3; j++) {
        System.out.print(i * j + "\t");
    }
    System.out.println();
}
// 1  2  3
// 2  4  6
// 3  6  9
✅Gebruik een for-lus als je op voorhand weet hoeveel keer je wilt herhalen. Gebruik een while-lus als dat pas tijdens de uitvoering bekend is.

Methoden

 ·  Herbruikbare blokken code

Opbouw van een methode

// toegang  returntype  naam       (parameters)
public static int     somVan      (int a, int b) {
    return a + b;
}

Methode zonder returnwaarde (void)

public static void drukAf(String tekst) {
    System.out.println(tekst);
}

// Oproepen:
drukAf("Hallo!");

Methode met returnwaarde

public static double berekenOppervlakte(double breedte, double hoogte) {
    return breedte * hoogte;
}

// Oproepen en opslaan:
double opp = berekenOppervlakte(5.0, 3.0); // opp = 15.0

Methode met boolean return

public static boolean isEven(int getal) {
    return getal % 2 == 0;
}

if (isEven(4)) {
    System.out.println("4 is even");
}

Overloading

Meerdere methodes met dezelfde naam, maar verschillende parameters:

public static int som(int a, int b) {
    return a + b;
}
public static int som(int a, int b, int c) {
    return a + b + c;
}
public static double som(double a, double b) {
    return a + b;
}
📌Een methode met static kan je oproepen zonder eerst een object te maken. In een klasse (OOP) laat je static weg.

Enumeratie (enum)

 ·  Vaste lijst van waarden

Wat is een enum?

Een enum is een speciaal type dat een vaste reeks benoemde waarden bevat. Ideaal voor categorieën zoals kleuren, seizoenen, statussen, enz.

Enum definiëren

public enum Seizoen {
    LENTE, ZOMER, HERFST, WINTER
}

Enum gebruiken

Seizoen huidig = Seizoen.ZOMER;

if (huidig == Seizoen.ZOMER) {
    System.out.println("Het is zomer!");
}

Enum in een switch

switch (huidig) {
    case LENTE:
        System.out.println("Bloemen bloeien");
        break;
    case ZOMER:
        System.out.println("Warm en zonnig");
        break;
    case HERFST:
        System.out.println("Bladeren vallen");
        break;
    case WINTER:
        System.out.println("Koud en donker");
        break;
}

Enum met extra info

public enum Dag {
    MAANDAG(true), DINSDAG(true), WOENSDAG(true),
    DONDERDAG(true), VRIJDAG(true),
    ZATERDAG(false), ZONDAG(false);

    private boolean werkdag;

    Dag(boolean werkdag) {
        this.werkdag = werkdag;
    }

    public boolean isWerkdag() { return werkdag; }
}

// Gebruik:
Dag vandaag = Dag.VRIJDAG;
System.out.println(vandaag.isWerkdag()); // true

Alle waarden doorlopen

for (Seizoen s : Seizoen.values()) {
    System.out.println(s);
}
// LENTE, ZOMER, HERFST, WINTER
✅Enum-waarden vergelijk je met == (niet .equals()). Dat is één van de weinige uitzonderingen in Java.

Arrays

 ·  Geordende reeks van waarden

Array aanmaken

// Methode 1: direct initialiseren
int[] scores = {85, 92, 78, 95, 61};

// Methode 2: lengte opgeven, later waarden toewijzen
int[] scores = new int[5];
scores[0] = 85;
scores[1] = 92;

// String-array
String[] namen = new String[3];
namen[0] = "Jan";

Array uitlezen

int[] getallen = {10, 20, 30, 40, 50};

System.out.println(getallen[0]);       // 10 — index start bij 0!
System.out.println(getallen.length);   // 5 — aantal elementen

Array doorlopen met for

int[] getallen = {10, 20, 30, 40, 50};

// Klassieke for-lus
for (int i = 0; i < getallen.length; i++) {
    System.out.println(getallen[i]);
}

// For-each (korter)
for (int g : getallen) {
    System.out.println(g);
}

Nuttige bewerkingen

int[] getallen = {10, 20, 30, 40, 50};

// Som berekenen
int som = 0;
for (int g : getallen) { som += g; }  // som = 150

// Maximum zoeken
int max = getallen[0];
for (int g : getallen) {
    if (g > max) max = g;
}
// max = 50

// Array sorteren
Arrays.sort(getallen); // {10, 20, 30, 40, 50} → gesorteerd
⚠️Een array heeft een vaste lengte. Als je dynamisch elementen wil toevoegen of verwijderen, gebruik dan een ArrayList!

OOP & Klassen

 ·  Basis van objectgeoriënteerd programmeren

Wat is een klasse?

Een klasse is een sjabloon/blauwdruk voor objecten. Ze bevat attributen (eigenschappen) en methodes (gedrag).

Structuur van een klasse

public class Toetsenbord {
    // 1. Attributen (private!)
    private String serienummer;
    private String merk;
    private boolean draadloos;
    private BigDecimal prijs;

    // 2. Constructor(s)
    public Toetsenbord(String merk) {
        this.merk = merk;
    }
    public Toetsenbord() {     // default constructor
        this.merk = "Logitech";
        this.draadloos = true;
    }

    // 3. Getters & Setters
    public String getMerk() { return merk; }
    public void setMerk(String merk) { this.merk = merk; }
    public boolean isDraadloos() { return draadloos; }

    // 4. Andere methodes
    public BigDecimal berekenPrijs(int aantal) {
        return prijs.multiply(BigDecimal.valueOf(aantal));
    }

    // 5. toString
    @Override
    public String toString() {
        String tekst = "";
        tekst += "Merk: " + merk + "\n";
        tekst += "Draadloos: " + draadloos + "\n";
        return tekst;
    }
}

Belangrijke begrippen

BegripUitleg
privateAttribuut enkel zichtbaar binnen de klasse
publicZichtbaar vanuit andere klassen
thisVerwijst naar het huidige object
getterMethode om attribuut op te vragen (getMerk())
setterMethode om attribuut in te stellen (setMerk())
toString()Geeft tekstuele weergave van het object
@OverrideOverschrijft een methode van de superklasse

Methode-overloading

Meerdere methodes met dezelfde naam maar verschillende parameters:

public BigDecimal berekenPrijs(int aantal) { ... }
public BigDecimal berekenPrijs(boolean draadloos) { ... }
✅Attributen altijd private maken en werken met getters/setters — dit noemen we encapsulatie.

Abstracte Klassen & Methodes

Wat is een abstracte klasse?

Een abstracte klasse is een klasse waarvan je geen objecten kunt aanmaken. Ze dient als basis voor subklassen.

public abstract class Klassenaam {
    // ... inhoud
}
⚠️new Klassenaam() is NIET mogelijk bij abstracte klassen!

Abstracte methode

Een abstracte methode heeft geen implementatie — subklassen zijn verplicht deze te implementeren.

public abstract returntype methodenaam(parameterlijst);

Regels

RegelUitleg
Alleen abstracte klassen mogen abstracte methodes bevattenEen gewone klasse kan geen abstracte methode hebben
Subklassen moeten alle abstracte methodes implementerenAnders is de subklasse ook abstract
In UML: cursief weergegevenAbstracte namen worden schuin gedrukt in UML-schema's

Voorbeeld

public abstract class Dier {
    private String naam;

    public abstract void maakGeluid(); // verplicht te implementeren
    
    public String getNaam() { return naam; }
}

public class Hond extends Dier {
    @Override
    public void maakGeluid() {
        System.out.println("Woef!");
    }
}

'Heeft-een' Relatie

 ·  Object als attribuut

Concept

Een "heeft-een" relatie betekent dat een klasse (parent) een object van een andere klasse (child) als attribuut heeft.

UML-regels

ElementUitleg
PijlJuiste pijl in de juiste richting (leesrichting: "heeft")
Multipliciteitminimum..maximum (bijv. 0..1, 0..*, 1..1)
RelatienaamNaam van de childklasse met kleine letter, voorafgegaan door -

Multipliciteit 0..1 of 1..1 (één child)

public class Auto {
    private Motor motor; // child object als attribuut

    public Motor getMotor() { return motor; }
    public void setMotor(Motor motor) {
        if (motor != null) // check bij 1..1
            this.motor = motor;
    }

    @Override
    public String toString() {
        return motor.toString(); // toString van child oproepen!
    }
}

Multipliciteit 0..* of meer (lijst van children)

public class Klaslokaal {
    private ArrayList<Tafel> tafelLijst;

    public Klaslokaal() {
        tafelLijst = new ArrayList<Tafel>(); // instantiëren in constructor!
    }
    public void voegTafelToe(Tafel tafel) { tafelLijst.add(tafel); }
    public boolean schrapTafel(String code) { /* zoek en verwijder */ }
    public Tafel vraagTafelOp(String code) { /* zoek en geef terug */ }
    public int geefAantalTafels() { return tafelLijst.size(); }
}
📌De toString() van de parentklasse moet de toString() van alle children oproepen via een lus!

Klaslokaal heeft Tafel

 ·  Concreet voorbeeld van heeft-een relatie

Klasse Tafel

public class Tafel {
    private String code;
    private int hoogteInCm;

    public Tafel(String code) { this.code = code; }

    public String getCode() { return code; }
    public int getHoogteInCm() { return hoogteInCm; }
    public void setHoogteInCm(int hoogte) { this.hoogteInCm = hoogte; }
}

Klaslokaal – sleutelmethodes

// Tafel toevoegen
public void voegTafelToe(Tafel tafel) {
    tafelLijst.add(tafel);
}

// Tafel opvragen op code
public Tafel vraagTafelOp(String code) {
    for (int i = 0; i < tafelLijst.size(); i++) {
        if (tafelLijst.get(i).getCode().equals(code)) {
            return tafelLijst.get(i);
        }
    }
    return null; // niet gevonden
}

// Tafel schrappen
public boolean schrapTafel(String code) {
    for (int i = 0; i < tafelLijst.size(); i++) {
        if (tafelLijst.get(i).getCode().equals(code)) {
            tafelLijst.remove(i);
            return true;
        }
    }
    return false;
}

// Aantal tafels
public int geefAantalTafels() { return tafelLijst.size(); }
⚠️Controleer altijd op null na vraagTafelOp() — als de tafel niet bestaat geeft de methode null terug!

JavaFX met Meerdere Schermen

Concept

In JavaFX kan je meerdere Stage-objecten aanmaken. Elk scherm is een aparte Stage.

Scherm aanmaken als aparte klasse

public class OmtrekBerekeningScherm {
    public static Stage maakOmtrekBerekeningStage(double straal) {
        Stage stage = new Stage();
        VBox root = new VBox(5);
        
        double omtrek = 2 * Math.PI * straal;
        Label label = maakLabel("Omtrek: " + omtrek);
        Button sluitKnop = maakSluitenKnop(stage);
        
        root.getChildren().addAll(label, sluitKnop);
        stage.setScene(new Scene(root, 550, 550));
        return stage; // stage teruggeven, NIET .show() hier!
    }
}

Scherm openen vanuit event handler

knop.setOnMouseClicked(new EventHandler<MouseEvent>() {
    @Override
    public void handle(MouseEvent e) {
        // Scherm aanmaken en tonen
        maakOmtrekBerekeningStage(straal).show();
    }
});

Sluitenknop

public static Button maakSluitenKnop(Stage stage) {
    Button knop = new Button("Sluiten");
    knop.setOnMouseClicked(e -> stage.close());
    return knop;
}
✅Elk scherm in een aparte klasse met een static methode die een Stage teruggeeft. Zo blijft de code overzichtelijk.

Werken met Datums

 ·  java.time.LocalDate

1. Datum aanmaken

LocalDate.now()                        // vandaag
LocalDate.of(2000, 8, 5)              // specifieke datum
LocalDate.parse("2006-06-12")         // uit String (yyyy-MM-dd)

2. Datum formatteren

datum.format(DateTimeFormatter.ofLocalizedDate(FormatStyle.SHORT))   // 27/01/26
datum.format(DateTimeFormatter.ofLocalizedDate(FormatStyle.MEDIUM))  // 27 jan 2026
datum.format(DateTimeFormatter.ofLocalizedDate(FormatStyle.LONG))    // 27 januari 2026
datum.format(DateTimeFormatter.ofLocalizedDate(FormatStyle.FULL))    // maandag 27 januari 2026

3. Rekenen met datums

LocalDate.now().plusDays(24)      // + 24 dagen
LocalDate.now().minusDays(24)     // - 24 dagen
LocalDate.now().plusWeeks(24)     // + 24 weken
LocalDate.now().plusMonths(24)    // + 24 maanden
LocalDate.now().plusYears(24)     // + 24 jaar

4. Info uit datum halen

LocalDate.now().getYear()         // huidig jaar
LocalDate.now().getMonth()        // maand (bv. JANUARY)
LocalDate.now().getDayOfMonth()   // dag van de maand
LocalDate.now().getDayOfYear()    // dag van het jaar
LocalDate.now().getDayOfWeek()    // weekdag (bv. MONDAY)
LocalDate.now().isLeapYear()     // schrikkeljaar? true/false

5. Verschil tussen datums

long aantalDagen = ChronoUnit.DAYS.between(
    LocalDate.of(2024, 7, 15),
    LocalDate.of(2026, 10, 2)
);

6. Datums vergelijken

datum1.isBefore(datum2)  // datum1 vroeger dan datum2?
datum1.isAfter(datum2)   // datum1 later?
// enum teruggeven: VOOR, NA, GELIJK

7. Volgende/vorige weekdag

datum.with(TemporalAdjusters.next(DayOfWeek.SUNDAY))          // eerstvolgende zondag
datum.with(TemporalAdjusters.previous(DayOfWeek.SUNDAY))      // vorige zondag
datum.with(TemporalAdjusters.nextOrSame(DayOfWeek.SUNDAY))     // zondag (vandaag of volgende)
datum.with(TemporalAdjusters.previousOrSame(DayOfWeek.SUNDAY))  // zondag (vandaag of vorige)

Werken met Bedragen (BigDecimal)

 ·  java.math.BigDecimal

Waarom BigDecimal?

double geeft afrondingsfouten bij geld. BigDecimal is nauwkeurig.

1. Aanmaken (altijd op 2 decimalen)

BigDecimal.valueOf(2).setScale(2, RoundingMode.HALF_UP)   // uit int
BigDecimal.valueOf(5.5).setScale(2, RoundingMode.HALF_UP)  // uit double
new BigDecimal("4.456").setScale(2, RoundingMode.HALF_UP)  // uit String
BigDecimal.ZERO.setScale(2, RoundingMode.HALF_UP)           // nul bedrag

2. Rekenen

bedrag1.add(bedrag2)                                      // optellen
bedrag1.subtract(bedrag2)                                 // aftrekken
bedrag1.multiply(bedrag2)                                 // vermenigvuldigen
bedrag1.divide(bedrag2, 2, RoundingMode.HALF_UP)          // delen

3. Vergelijken

int res = bedrag1.compareTo(bedrag2);
// -1 = KLEINER, 0 = GELIJK, 1 = GROTER
✅Gebruik altijd RoundingMode.HALF_UP voor geldbedragen. Nooit gewoon divide() zonder afrondingsmodus — dat geeft een exception bij oneindige decimalen!

JDBC: Prepared Statements

Wat is een prepared statement?

Een geprecompileerde SQL-instructie met ? als placeholders voor parameters.

✅Voordelen: makkelijk te parametriseren + sneller door precompilatie

Aanmaken

PreparedStatement ps = connectie.prepareStatement(
    "SELECT * FROM employees WHERE salary > ? AND department = ?"
);
// ? zijn placeholders — nummering start bij 1!

Parameters instellen

ps.setDouble(1, 80000);         // eerste ? = salary
ps.setString(2, "Legal");      // tweede ? = department

Uitvoeren

// Query (SELECT)
ResultSet resultaat = ps.executeQuery();

// Update (INSERT / UPDATE / DELETE)
int aantalRijen = ps.executeUpdate();
⚠️Parameters zijn one-based! Begin te tellen vanaf 1, niet 0!

Parametermethodes

MethodeType

setInt(nr, waarde)int
setDouble(nr, waarde)double
setBinaryStream(nr, stream)BLOB

JDBC: Transacties

Wat is een transactie?

Een reeks SQL-instructies die allemaal of geen van allen uitgevoerd worden.

TermUitleg
commitTransactie geslaagd → wijzigingen opslaan voor alle gebruikers
rollbackFout opgetreden → alles ongedaan maken
autocommitStandaard AAN: na elke SQL automatisch commit

Autocommit uitschakelen

connectie.setAutoCommit(false); // verplicht bij transacties!

Commit of rollback

try {
    connectie.setAutoCommit(false);
    statement.executeUpdate("DELETE FROM employees WHERE department='HR'");
    statement.executeUpdate("UPDATE employees SET salary=300000 WHERE...");

    if (goedgekeurd) {
        connectie.commit();    // alles opslaan
    } else {
        connectie.rollback(); // alles ongedaan maken
    }
} catch (Exception e) {
    connectie.rollback(); // bij fout: rollback!
}
⚠️Zolang geen commit() gedaan is, zijn de wijzigingen voor andere gebruikers niet zichtbaar!

JDBC: ResultSet-metadata

Wat?

Informatie opvragen over de kolommen in het resultaat van een SQL-query.

ResultSetMetaData ophalen

ResultSet resultaat = statement.executeQuery("SELECT id, last_name FROM employees");
ResultSetMetaData meta = resultaat.getMetaData();

Informatie over kolommen

meta.getColumnCount()           // aantal kolommen
meta.getColumnName(teller)      // naam van kolom nr. teller
meta.getColumnTypeName(teller)  // type (bv. "VARCHAR", "INT")
meta.isAutoIncrement(teller)    // auto-increment? true/false

Itereren over kolommen

// Kolomnummering is ONE-BASED (start bij 1!)
for (int i = 1; i <= meta.getColumnCount(); i++) {
    System.out.println(meta.getColumnName(i));
}
⚠️Kolomnummering begint bij 1, niet bij 0!

JDBC: BLOB's

Wat is een BLOB?

Binary Large Object — binaire data zoals afbeeldingen of muziek.

TypeMax grootte
TINYBLOB255 bytes
BLOB64 KB
MEDIUMBLOB16 MB
LONGBLOB4 GB

BLOB opslaan

1. PreparedStatement aanmaken met "?" als placeholder:
PreparedStatement ps = connectie.prepareStatement(
    "UPDATE employees SET resume=? WHERE email='john@foo.com'"
);

2. BLOB-parameter instellen via FileInputStream:
File bestand = new File("afbeelding.jpg");
FileInputStream invoer = new FileInputStream(bestand);
ps.setBinaryStream(1, invoer); // 1 = eerste ?

3. Uitvoeren:
ps.executeUpdate();

JDBC: CLOB's

Wat is een CLOB?

Character Large Object — grote tekstbestanden in de databank. In MySQL: LONGTEXT kolom.

CLOB opslaan

1. PreparedStatement aanmaken:
PreparedStatement ps = connectie.prepareStatement(
    "UPDATE employees SET resume=? WHERE email='john@foo.com'"
);

2. CLOB-parameter instellen via FileReader:
File tekstbestand = new File("tekst.txt");
FileReader invoer = new FileReader(tekstbestand);
ps.setCharacterStream(1, invoer); // let op: CharacterStream!

3. Uitvoeren:
ps.executeUpdate();
